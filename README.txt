Ionic 4, Angular 8 and Cordova jewellery Managment System Example
This source code is part of Ionic 4, Angular 8 and Cordova jewellery Managment SystemExample tutorial login page, Jewelley calculation and billing page with pdf.

To run in the real device:

$ Clone this repo
$ Run npm install
$ Run ionic cordova platform rm ios
$ Run ionic cordova platform add ios
$ Run ionic cordova platform rm android
$ Run ionic cordova platform add android

Plugin Details:
Camera
$ ionic cordova plugin add cordova-pdf-generator
$ npm install @ionic-native/cordova-pdf-generator
$ ionic cordova plugin add cordova-plugin-file
$ npm install @ionic-native/ordova-plugin-file
$ ionic cordova plugin add cordova-plugin-file-opener2
$ npm install @ionic-native/cordova-plugin-file-opener2

Run the project:

$ Run ionic cordova run ios
$ Run ionic cordova run android
